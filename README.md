# claude-skills

A Claude Code plugin marketplace.

## Install

```
/plugin marketplace add CJHwong/claude-skills
```

Then install the plugins you want:

```
/plugin install agents-md-skill@claude-skills
/plugin install sort-later@claude-skills
```
